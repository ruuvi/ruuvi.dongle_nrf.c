◆ app_ble_scan_start()

rd_status_t app_ble_scan_start ( void  )

Start a scan sequence.

Checks if any PHY is enabled. Runs through enabled modulations and channels, automatically starting on next after one has finished. Calls given scan callback on data. If all PHYs are disabled, calls app_ble_scan_stop() to stop scanning process until any PHY is reactivated.

Return values
RD_SUCCESSon success.
